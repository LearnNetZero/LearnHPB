SUPPLY FAN SPEED CONTROLLER INTEGRATOR TIME CONSTANT

Associated System: Fan

The integrator time constant is the constant multiplier on the integral component of a proportional-integral (PI) controller. A PI controller is a control loop feedback mechanism widely used in building control systems. The controller calculates the difference between a measured value and a desired set-point value, known as the error, and seeks to minimize this error by adjusting the control output signals though an algorithm that considers the present value of the error (proportional component), and an accumulation of past errors (integral component). The integrator time constant is the constant that is multiplied by the integration of the error over a previous amount of time to make up the integral component of the control output signal.

For a supply fan, the fan motor speed is the controlled variable. The fan motor receives the control signal and adjusts the motor speed, which in turn is monitored by the control system until it achieves the correct position within acceptable error limits.

This value determines the integral response of the controller . If this value is set too high, it may cause instability in the system by overshooting.

The value of this variable can be adjusted during a simulation run-time.
