ZONE 9 VAV REHEAT COIL CONTROLLER P GAIN

Associated System: VAV Boxes

The P gain is the constant multiplier on the proportional component of a proportional-integral (PI) controller. A PI controller is a control loop feedback mechanism widely used in building control systems. The controller calculates the difference between a measured value and a desired set-point value, known as the error, and seeks to minimize this error by adjusting the control output signals though an algorithm that considers the present value of the error (proportional component), and an accumulation of past errors (integral component). The P gain is the constant that is multiplied by the present value of the error to make up the proportional component of the control output signal.

For a reheat water coil valve, the valve position is the controlled variable. The valve actuator receives the control signal and adjusts the valve position, which in turn is monitored by the control system until it achieves the correct position within acceptable error limits.

This value determines the linear response of the controller . If this value is set too high, it may cause instability in the system by overshooting.

The value of this variable can be adjusted during a simulation run-time.
