ZONE 1 (NORTH) HEATING SET POINT

Associated System: Zones

Zone heating set points are the temperature that an HVAC system tries to maintain within a partcular zone while in heating mode.

Increasing this value will increase the amount heating required in order for the zone temperature to match the set point. As a consequence, the HVAC energy demand will increase.

The value of this variable can be adjusted during a simulation run-time.
