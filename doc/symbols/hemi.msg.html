<static> {hemi.dispatch.MessageTarget} hemi.msg.subscribe(type, handler, opt_func, opt_args)
Register the given handler to receive Messages of the specified type from any source. This creates a MessageTarget.
Parameters:
{string} type
type of Message to handle
{Object} handler
either a function or an object
{string} opt_func
name of the function to call if handler is an object
{string[]} opt_args
optional array of names of arguments to pass to the handler. Otherwise the entire Message is just passed in.
Returns:
{hemi.dispatch.MessageTarget} the created MessageTarget
<static> {hemi.dispatch.MessageTarget} hemi.msg.unsubscribe(target, opt_type)
Remove the given MessageTarget for the specified Message type. Note that this removes a MessageTarget registered with the wildcard as the source id. It does not remove the MessageTarget from any Citizens it may be directly registered with.
Parameters:
{hemi.dispatch.MessageTarget} target
the MessageTarget to remove from the Dispatch
{string} opt_type
Message type the MessageTarget was registered for
Returns:
{hemi.dispatch.MessageTarget} the removed MessageTarget or null
