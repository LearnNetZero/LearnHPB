Class hemi.handlers.ValueCheck


Extends hemi.world.Citizen.
A ValueCheck handler checks a set of values against a specified set of values from the Message to handle. If the values all match, the Message is passed to the actual handler.
Defined in: <hemi/handlers/valueCheck.js>.
