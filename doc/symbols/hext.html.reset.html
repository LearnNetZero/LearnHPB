Namespace hext.html.reset

A set of functions for setting up Reset behavior and displaying a button that allows the user to reset the World. In order for the Reset button to be displayed, you need to have a div with the container id in your viewer page. The default container id that Reset searches for is 'hext-reset', but you may change the id if you wish.
Defined in: <hext/html/reset.js>.
