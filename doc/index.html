Class Index

_global_

hemi

The core Hemi library used by Kuda.

hemi.animation

A module for animating hemi.model.Models.

hemi.animation.Animation

An Animation contains a target to animate, a begin time, an end time, and hemi.animation.Loops for repeating sections of the Animation.

hemi.animation.Loop

A Loop contains a start time and stop time as well as the number of iterations to perform for the Loop.

hemi.audio

A module for supporting audio playback with the flash_mp3_player_js found at http://flash-mp3-player.net an open source player.

hemi.console

A module for displaying log, warning, and error messages to a console element on a webpage.

hemi.core

A module for handling low level functionality and wrapping calls to the underlying O3D library.

hemi.curve

A module for curves and particle systems.

hemi.curve.Curve

A Curve is used to represent and calculate different curves including: linear, bezier, cardinal, and cubic hermite.

hemi.curve.Particle

A Particle allows for the movement of a transform along a set of points.

hemi.curve.ParticleSystem

A ParticleSystem manages a set of Particle objects, and fires them at the appropriate intervals.

hemi.dispatch

A module for message dispatching and handling.

hemi.dispatch.Message

A Message is sent whenever an event occurs.

hemi.dispatch.MessageSpec

A MessageSpec specifies a certain Message type and source and contains a set of MessageTargets that have registered to receive Messages with matching specs.

hemi.dispatch.MessageTarget

A MessageTarget registers with a MessageSpec to receive Messages that match its attributes.

hemi.effect

A module for generating particle effects.

hemi.effect.Burst

A Burst generates one set of particles at a time.

hemi.effect.Effect

An Effect is a 3D particle effect.

hemi.effect.Emitter

An Emitter constantly generates particles.

hemi.effect.ParticleFunction

A ParticleFunction specifies a predefined per-particle parameter setting function and any properties it might require.

hemi.effect.Trail

A Trail is a particle effect that can be started and stopped like an animation.

hemi.handlers

A module for prebuilt Message handlers that perform common Message handling tasks more easily.

hemi.handlers.ValueCheck

A ValueCheck handler checks a set of values against a specified set of values from the Message to handle.

hemi.hud

A module for heads up display (HUD) creation and manipulation.

hemi.hud.HudButton

A HudButton contains textures for different images based on if the button is enabled or if a mouse is hovering over it.

hemi.hud.HudDisplay

A HudDisplay contains one or more HudPages to display sequentially.

hemi.hud.HudElement

A HudElement contains the basics of any element to be drawn on the canvas.

hemi.hud.HudImage

A HudImage contains a texture and display options for a single image on the HUD.

hemi.hud.HudManager

A HudManager creates the appropriate view components for rendering a HUD.

hemi.hud.HudPage

A HudPage contains other HudElements and display options for drawing a single page on the HUD.

hemi.hud.HudText

A HudText contains formated text and display options for a single area of text on the HUD.

hemi.hud.Theme

A Theme contains configuration options for displaying HUD elements like pages and text.

hemi.input

A module for handling all keyboard and mouse input.

hemi.loader

A module for loading resources (Octane, models, images, etc).

hemi.manip

A module for defining draggable objects.

hemi.manip.Draggable

A Draggable allows a 3d object to be dragged around the scene with the mouse, constrained to a defined 2d plane.

hemi.manip.Turnable

A Turnable allows a transform to be turned about an axis by the user clicking and dragging with the mouse.

hemi.model

A module for managing 3D models and their assets.

hemi.model.Model

A Model is a Javascript representation of a 3D model.

hemi.model.ModelConfig

A ModelConfig contains the pack, transform, and animation time attributes for a Model.

hemi.model.TransformUpdate

A TransformUpdate allows changes to a Transform in a Model to be persisted through Octane.

hemi.motion

A module for moving and rotating objects in the scene.

hemi.motion.Rotator

A Rotator makes automated rotation easier by allowing simple calls such as setVel to begin the automated spinning of a transform.

hemi.motion.Translator

A Translator provides easy setting of linear velocity and acceleration of shapes and transforms in the 3d scene.

hemi.msg

A module for managing the string literals for Message types.

hemi.octane

A module for creating objects (including Worlds) from Octane.

hemi.picking

A module for picking shapes in a 3D space from a mouse click.

hemi.picking.HemiTransformInfo

A HemiTransformInfo extends O3D's TransformInfo to allow the user to specify if the referenced Transform should be pickable or not.

hemi.scene

A module for grouping world events and behavior into logical scenes.

hemi.scene.Scene

A Scene represents a logical grouping of behavior, events, and interactions.

hemi.shape

A module for easily creating primitives in Kuda.

hemi.shape.Shape

A Shape is a wrapper class around basic geometric shapes such as cubes and spheres that allows them to interact with the World in complex ways.

hemi.sprite

A module for defining animated sprites and billboards.

hemi.sprite.Sprite

A Sprite can display a 2d image on a plane with several options.

hemi.texture

A module for texture set management, specifically designed to handle shader sampling and texture maps.

hemi.texture.TextureSampler

A TestureSampler contains a textureURL and a value.

hemi.texture.TextureSet

A TextureSet can manage hemi.texture.TextureSampler's that are part of a set as defined by the author.

hemi.utils

A module to provide various utilities for Hemi.

hemi.utils.Hashtable

A Hemi Hashtable extends a Hashtable to allow it to be queried for Object attributes that are not the Hash key.

hemi.view

A module for defining camera and viewpoint options.

hemi.view.Camera

A Camera contains all of the properties and functionality for viewing a 3D scene.

hemi.view.ClientSize

A ClientSize contains the height and width of the O3D client and updates camera projections when either value changes.

hemi.view.Viewpoint

A Viewpoint describes everything needed for a view - eye, target, up axis, field of view, near plane, and far plane.

hemi.world

A module for managing all elements of a 3D world.

hemi.world.Citizen

A Citizen is a uniquely identifiable member of a World that is able to send Messages through the World's dispatch.

hemi.world.TransformOwner

A TransformOwner contains a Citizen that owns at least one transform and entries of other Citizens that need to be given any of those transforms as they become available during Octane loading.

hemi.world.TransformRegistry

A TransformRegistry maintains listings of which transforms to distribute to which Citizens as they become available during Octane loading.

hext

Hext (short for Hemi Extensions) is a library of extensions to the Hemi engine developed by the Kuda team.

hext.editor

A module for simulation engines.

hext.engines

A module for simulation engines.

hext.engines.Location

A Location represents a discrete volume of space with a distinct pressure value.

hext.engines.Portal

A Portal represents an opening between two Locations that air can flow through.

hext.engines.PressureEngine

A PressureEngine calculates pressure changes over time across a system of Locations connected by Portals.

hext.house

A module for pre-defined household objects, with standard behaviors attached.

hext.house.Door

Door is a standard door that swings 90 degrees when clicked on.

hext.house.Window

Window is a pre-defined window that slides to a given point when clicked on

hext.html

A module for pre-built HTML containers and elements that interact with the 3D scene.

hext.html.hint

A set of functions for creating hint messages and displaying a button that allows the user to show/hide those messages.

hext.html.reset

A set of functions for setting up Reset behavior and displaying a button that allows the user to reset the World.

hext.html.toolbar

A set of functions for displaying ToolbarViews.

hext.html.toolViews

A set of functions for displaying HtmlViews.

hext.hud

A module for additional HUD features.

hext.hud.PagingInfo

A PagingInfo contains text and images for displaying information about a multi-page HudDisplay.

hext.msg

A module to extend the Message types in Hemi to support new features in Hext.

hext.tools

hext.tools.BaseController

hext.tools.BaseTool

hext.tools.BlowerDoor

hext.tools.HtmlView

hext.tools.HtmlViewConfig

hext.tools.Manometer

hext.tools.ManometerToolbarView

hext.tools.ManometerToolbarViewConfig

hext.tools.ManometerTube
