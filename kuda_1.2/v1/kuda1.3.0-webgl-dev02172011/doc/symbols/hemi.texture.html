Namespace hemi.texture

A module for texture set management, specifically designed to handle shader sampling and texture maps. An o3d.Material will have a parameter that is a sampler when a material is using a texture. The paramter is either an emmissive, or diffuse sampler. This module makes easy to swap textures dynamically on hemi.model.Models. See the lighting sample.
Defined in: <hemi\texture.js>.
