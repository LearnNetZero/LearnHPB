Migrating from 1.2.0 to 1.3.0

1) Kuda now runs off of WebGL rather than the O3D plugin. This will first
require you and your users to run the latest version of Chrome, Firefox, or
Safari. In addition to converting your o3dtgz assets, the following changes will
be necessary to convert your pre-existing files:

- in your .html files, add this script include:
	<script type="text/javascript" src="../o3d-webgl/base.js">
	</script>
- in the matching .js file, replace:
	o3djs.util.makeClients(function(clientElements) {
  with:
	o3djs.webgl.makeClients(function(clientElements) {
- when loading models, replace:
	myModel.setFileName('assets/house_v12.o3dtgz');
  with:
	myModel.setFileName('assets/house_v12/scene.json');

2) Hext paging extension has changed slightly:
---------- Old code ----------
hext.hud.showPagingInfo();
var display = new hemi.hud.HudDisplay();
display.show();

---------- New code ----------
var display = new hemi.hud.HudDisplay();
hext.hud.addPagingInfo(display);
display.show();
